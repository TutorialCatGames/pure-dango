# pure-dango v1.00

A simple language built from JavaScript.

## Quick Start
``` bash
cd path to pure-dango
node src/index.ts run example.pds

```

## Make EXE:
use src/scripts/installer.bat (warning this will close file explorer)
then run src/scripts/build.js

## Language Guide
For full pure-dango language details, see [language details](https://tutorialcatgames.github.io/pure-dango/)
